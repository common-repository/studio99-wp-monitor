This plugin provides to Studio99 Wordpress Monitoring Console informations about the web site.

== Description ==

Studio99 is a web agency which manage hundres of website.
In order to allow staff from Studio99 to keep monitored the web agency websites, this plugin provides general informations about Wordpress to Studio99 Wordpress Monitoring Console.

This plugin will provide informations about:

- Apache
- MySQL
- Wordpress Core Version
- Template Used
- Plugins installed

This plugin can only provide the previous informations only to Studio99 Wordpress Monitoring Console.
Studio99's staff will keep monitored their web sites and they will proceed with updates, upgrades or any other important operation.
